# Pi-Router-Script
This uses a variety of functions to create a wireless router out of a Raspberry Pi.
# Features
* DNS Caching and Ad Blocking
* DNS over HTTPS with Cloudflare's DNS, [1.1.1.1](https://1.1.1.1)
* OpenVPN Server
* OpenVPN Client
* Dynamic DNS Host
* Local Certificate Authority
